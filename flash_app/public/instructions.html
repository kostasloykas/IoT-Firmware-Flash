Instructions for Linux:

  1. The user must grant read and write access to the right port (e.g sudo chmod 666 /dev/ttyUSB0)
  2. USB permmissions sudo usermod -aG dialout sudo nano /etc/udev/rules.d/99-slusb.rules and append this command ' SUBSYSTEM=="usb", MODE="0666", GROUP="dialout" '
  3. Run google chrome as root: $sudo google-chrome --user-data-dir --test-type --no-sandbox
