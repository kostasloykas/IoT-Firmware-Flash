Flash+

Flash+ is a web-based application that contributes to the advancement of IoT device management by providing a practical solution for firmware updates. The system is designed to interface with popular embedded operating systems such as Contiki-NG and Zephyr OS's, leveraging their bootloader functionalities of device for firmware deployment. Through a user-friendly web interface, device administrators can effortlessly initiate the update process by selecting the target IoT device and specifying the firmware to be flashed in memory. Flash+ offers a secure firmware installation process leveraging the Signascribe script, an implementation developed by us. This script empowers users to create customized firmwares with integrated security features, including signature verification and certificate chain validation, ensuring a secure and trustworthy flashing process.

Supported Devices

Zolertia CC2538
Openmote-b
nRF 52840 Dongle
nRF 52840 Development Kit

Instructions for Linux:

  1. The user must grant read and write access to the right port (e.g sudo chmod 666 /dev/ttyUSB0)
  2. USB permmissions:
    $sudo usermod -aG dialout
    $sudo nano /etc/udev/rules.d/99-slusb.rules and append this command ' SUBSYSTEM=="usb", MODE="0666", GROUP="dialout" '
  3. Run google chrome as root: $sudo google-chrome --user-data-dir --test-type --no-sandbox

Instructions for Windows:

  1. Install the device's drivers if they are required. For cc2538 devices use Driver Booster and for nRF 52840 use Zadig.
  2. Run chrome as administrator.
